函数表达式不能以function开头

```javascript
function foo() {
    if(false) {
        function x() {};
    }
    return x;
}
console.log(foo());   //undefined
```
尽量别在if等非函数块中使用函数声明
```javascript
function foo() {
    if(true) {
        function x() {};
    }
    return x;
}
console.log(foo());   //[Function: x]
```
